# Assembly MIPS - Exercícios Progressivos

## 📖 Objetivo

Consolidar os conhecimentos em Assembly MIPS por meio de exercícios com dificuldade progressiva, abrangendo desde conceitos básicos até desafios mais complexos.

## 📂 Conteúdo

O repositório contém exercícios sobre:

- Estruturas de programação;
- Manipulação de registradores;
- Operações aritméticas;
- Resolução de problemas com dificuldade crescente.

## 💻 Tecnologias

- Assembly MIPS
- MARS Simulator

## ▶️ Como executar

1. Abra o arquivo no MARS Simulator.
2. Monte o programa (Assemble).
3. Execute o código (Run).

## 🎓 Disciplina

Arquitetura e Organização de Computadores – Fatec Zona Leste
